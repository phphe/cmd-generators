    Move files or folders

    Basic usage

    $ mv file_path target_path

    File path or pattern, example: Single file or folder: cat.jpg Relative path: ../photos/cat.jpg Absolute path: /photos/cat.jpg All files in current folder: ./*.* All jpg files in current folder: ./*.jpg All jpg files in current folder and its subfolders: ./**/*.jpg
    If target path no file or directory existing, move source to there and rename; If target is directory, move into the directory; If target is file, overwrite the file.
