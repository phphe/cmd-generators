    tarCMD Generator

    Saves many files together into a archive, and can restore individual files from the archive.

    • Create new archive
    • Add file into an existing archive
    • Add newer file into an existing archive
    • List archive contents
    • Extract files and directories from an archive
    • Find differences between archive and specified files or folders
    • Delete files or folders from an archive

    Create new archive

    $ tar -c -f file_path file_path

    List archive contents

    $ tar -t -f file_path

    Extract files and directories from an archive

    $ tar -x -f file_path
    ​help_outline

    Find differences between archive and specified files or folders

    $ tar -d -f file_path file_path
    ​help_outline

    Delete files or folders from an archive

    $ tar --delete -f file_path file_path
